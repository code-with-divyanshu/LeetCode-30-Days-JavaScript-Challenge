## ðŸ’¡ Approach

### **Approach 1: Using a simple `for` loop and an object**

```js
Array.prototype.groupBy = function (fn) {
  let obj = {};
  for (let i = 0; i < this.length; i++) {
    let key = fn(this[i]);
    if (obj[key] === undefined) {
      obj[key] = [this[i]];
    } else {
      obj[key].push(this[i]);
    }
  }
  return obj;
};
```
